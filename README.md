# Single Lever Faucet Casting Die

## Overview

This project showcases the design of a casting die for a single lever sanitary faucet developed in SolidWorks.

The objective of this project was to demonstrate practical CAD modeling and die design skills used in the sanitary products industry.

**Note:** Dimensions and manufacturing specifications are intentionally omitted. This repository is intended to showcase engineering design capabilities and CAD expertise.

## Design Process

1. Studied the faucet body geometry.
2. Developed the casting die concept.
3. Designed the core and cavity.
4. Created the complete die assembly.
5. Verified alignment and assembly fit.
6. Prepared the model for manufacturing visualization.
